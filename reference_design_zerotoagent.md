# zerotoagent.com — Full Design Reference
**URL:** https://www.zerotoagent.com/
**Company:** Zero to Agent — Toronto-based AI transformation consultancy
**Target market:** Enterprises >$25M revenue

---

## Aesthetic / Mood
Premium minimalism with an operator attitude. White-forward, clean, and confident — not flashy, not dark. Feels like a serious boutique firm that builds things. The design avoids "consulting theatre" aesthetics (no dense slide decks, no corporate navy). It reads as a startup founder's consultancy: fast, opinionated, results-oriented. Occasional animated GIFs (hero figure, goggles) inject personality without sacrificing cleanliness.

## Typography

- **Headlines:** Large, bold, geometric sans-serif. Short and punchy ("Win the next decade.")
- **Subheads:** Medium weight sans-serif, supporting context
- **Body:** Smaller sans-serif, readable, conventional
- **Tone:** No editorial serif/italic contrast, no monospace — clean and consistent throughout
- No font CDN links confirmed in source; likely system or bundled sans-serif

---

## Layout

- **Structure:** Vertical scrolling narrative — hero → 3-column services → case study cards → FAQ accordion → footer
- **Hero:** Full-width, centered headline + subhead, animated illustration (GIF)
- **Services:** Three columns: Strategy / Transformation / Engineering
- **Case studies:** Card grid showing client metrics ($50k saved, 100+ hrs, etc.) with logos
- **FAQ:** Accordion/expandable sections
- **CTA:** Persistent "Book a working session" → Calendly link
- **Grid:** Clean, padded sections; generous whitespace; no exposed grid lines

---

## Brand Voice (informs design decisions)

Key phrases that signal design tone:
- *"Toronto-based agentic engineers for operators who need leverage, not theatre"*
- *"No 6-month strategy work. No 200-slide presentations."*
- *"AI isn't scary. Ignoring it is."*
- *"We don't care about today. We care about the next decade."*

Voice: Direct, builder-brained, anti-bloat. Design should feel the same — no decorative excess.

---

## Icon / Illustration System

Custom SVG "tenex" icon set with outlines of: brain, hammer, globe, goggles, arrowhead. Paired with 3D statue/figure renders in the hero. Functional-meets-sculptural — not flat icons, not illustrative cartoons. Creates visual depth and modern sophistication.

---

## Component Notes

- **Buttons:** Clean, rounded, action-oriented ("Book a working session")
- **Cards:** Modular, metric-forward (outcome numbers prominent), logo-integrated
- **Nav:** Minimal — brand name + primary CTA
- **Footer:** Contact email (hello@zerotoagent.com), simple
